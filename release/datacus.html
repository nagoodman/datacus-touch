Datacus is a multitouch interface for viewing and exploring data. It runs on Windows 7 within Flash and uses the latest APIs to present information in a meaningful and interactive experience.

This website contains all of the papers, man pages, and presentations for Datacus. It also contains all source files for the Datacus program.
